# 萌娘DB

**注意：本项目是个人发起，不是萌娘百科的一部分**

从[萌娘百科](https://zh.moegirl.org.cn/)与[Bangumi](https://bgm.tv/)爬取并整理的数据。

||截止日期|
|---|---|
|萌娘百科分类爬虫|2025-12-28|
|萌娘百科wikitext爬虫|2025-12-28|
|bangumi|2025-12-23|

友情提醒：可通过`git clone --depth 1 https://github.com/Zzzzzzyt/moegirl-dataset.git`仅下载HEAD，减少下载量。

## 自行爬取

萌百wikitext爬虫（`src/moegirl/crawler/crawler_extra.py`）需要登陆访问修改页面api。请将萌娘百科登陆cookie置于`.env`，格式如下：

```text
MOEGIRL_COOKIES="moegirlSSOUserName=****; moegirlSSOUserID=****; moegirlSSOToken=****; moegirlSSO_session=****"
```

然后

```bash
make clean_all
make all
```

即可。

如中途出现网络错误，直接重新`make all`，脚本会自行断点重续。

萌百分类页爬虫大约需要6小时，萌百wikitext爬虫大约需要40小时。bangumi数据来自[dump](https://github.com/bangumi/Archive)，大约需要下载350MB。

## 萌百部分

所有人物、属性的key均为MediaWiki page title，全局唯一，可能会与显示的文章标题不一致。

### moegirl/attrs.json

爬取[Category:虚拟人物](https://zh.moegirl.org.cn/Category:虚拟人物)页面及其子分类页面的原始数据。

### moegirl/subjects.json

爬取[Category:各地区作品](https://zh.moegirl.org.cn/Category:各地区作品)页面及其子分类页面的原始数据。
